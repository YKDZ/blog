# 用焖烤法代替慢炖可以提高生活幸福感

焖烤法可以完全消除小火慢炖肉类食材时频繁抄底搅拌的工作量，让慢炖更轻松。

我偶尔会大批量地做台式卤肉和意式番茄肉酱，吃不完的就冻在冰箱里当作 “小帮手”，懒得做饭的时候就拿出来拌饭拌面，十分好用省事。

但这类高糖高蛋白质且需要长时间小火慢炖的料理，在炖煮阶段都需要偶尔搅拌，否则就会糊底，且时间越长需要的抄底频率就越高，一不小心就毁了一锅肉。

为了解决此问题，我探索出了使用焖烤法（Braising）来代替小火慢炖的方法。所谓焖烤法，就是将需要长时间慢炖的食材放到 **珐琅锅** 内，然后 **带盖** 直接放到提前预热到 160°C 左右的烤箱中层（烤网上）即可。最爽的是，一旦食材进了烤箱，在正式出锅前你就 **再也不用** 管它了，只要锅内还有液体，它就基本不会糊底或糊边（在做番茄肉酱时我尝试烤过五个小时以上）。因此我可以真正的 “慢炖”，直到食材达到我期望的状态为止。

至于原理，我觉得不管是再小的火，毕竟也还是激烈的单点 / 单面加热，锅底局部热流密度太高，而浓稠酱汁中的糖、蛋白质、肉末、番茄固形物、胶质会沉积或贴附在锅底；一旦局部水分不足，这些固形物就会迅速焦化甚至碳化。

但是焖烤不一样，热空气的热流密度远低于火焰，会将整个锅锅体乃至锅盖都均匀地升温至一个适中的温度，接触食材的那一面很难到达 200°C 以上，也就不会糊锅。

还有一个好处，由于锅盖也被加热了，所以食材中蒸发出的水蒸气不会像正常一样凝结在锅盖上再落回食材中，而是始终保持蒸汽状态，一部分留在锅内，一部分逸散到烤箱中（途中打开烤箱会有大量蒸汽，小心不要被烫伤），因此一是可以在全程创造一个高湿环境避免食材表面发干，二是可以减缓失水速度，让一锅食材可以承受住更久的慢炖，也就让卤肉和肉酱可以更加软烂融合。实际操作中我最后揭开锅盖时，内面一般都是没有任何水珠的。

这种做法也可以自然延伸到所有需要慢炖且粘稠、高糖、高蛋白质而极易糊底的料理，比如炖牛肉、咖喱之类。

当然这种做法也有坏处，就是这种均匀的加热很难做出大火收汁的效果，所以若有需要则还需要重新上灶开火来完成。当然我觉得这点坏处和之前节省的时间精力一比不值一提。

想象下午把红葱酥炸好、五花肉切好焦化好，一股脑丢进锅中，调好味之后盖上盖子直接放进烤箱，就可以两手一摊做甩手掌柜，等着晚上吃肉了。我想这就是所谓幸福生活吧。
